# Vrinda-Store-Data-Analysis - 2022
Excel Dashboard Project | Retail Insights | Sales & Customer Analysis

# Project Overview
This project analyzes Vrinda Store's 2022 sales and customer data using Excel, uncovering trends that help optimize marketing, sales strategy, and customer targeting.
The final output is an interactive dashboard showing performance across gender, channels, states, categories, and more.

# Key Business Questions Answered
1) Compare the sales and orders using single chart
2) Which month got the highest sales and orders?
3) Who purchased more- men or women in 2022?
4) What are different order status in 2022?
5) List top 10 states contributing to the sales?
6) Relation between age and gender based on number of orders
7) Which channel is contributing to maximum sales?
8) Highest selling category?, etc.
